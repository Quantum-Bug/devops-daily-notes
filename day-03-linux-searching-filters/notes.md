### Pipes
Send output of one command into another:

ls -l | grep ".txt"

### Redirection

| Symbol | Purpose |
|--------|---------|
| > | Overwrite output |
| >> | Append output |
| < | Input redirection |

Example:
echo "DevOps" > file.txt  
echo "Linux" >> file.txt
