Linux is a multi-user, multitasking operating system.

### Important Directories
| Directory | Purpose |
|----------|--------|
| / | Root directory |
| /home | User home folders |
| /etc | Config files |
| /var | Logs |
| /bin | Basic commands |
| /usr | User binaries |

Everything in Linux is treated as a file.
